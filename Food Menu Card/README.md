# The Hungry Spot - Digital Menu Card

Modern, responsive food menu card with:
- Cart system
- WhatsApp ordering
- UPI QR code payment
- Order number generation
- Customer loyalty points
- Beautiful food photos

## Features
- Add/remove items
- Real-time cart update
- WhatsApp order with pre-filled message
- Dynamic UPI QR for payment
- Loyalty points (1 point per ₹100)
- Mobile friendly design

## How to use
1. Clone repo
2. Open `index.html` in browser
3. Replace UPI ID and WhatsApp number in `js/script.js`
